Convergepoint disclosure form portal UI





This project was done over the span of 2 days. With a goal of this web application is to develop a user-friendly and easily understandable User Interface for the ConvergePoint Disclosure Portal. The portal is a software application used by employees of companies to complete annual disclosures and access relevant information and instructions. The design thinking approach I used is a human-centered methodology that focuses on understanding user needs, brainstorming creative solutions, and constantly reworking the prototype based on those solutions from user feedback. By using this approach, I aim to create a UI that targets to the specific needs of the disclosure process for all employees.
